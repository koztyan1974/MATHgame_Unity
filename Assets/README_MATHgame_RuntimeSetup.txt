MATHgame - Quick Start
======================

What is already configured:
- Runtime auto-bootstrap creates:
  - GameManager, UIManager, QuestionManager
  - Player and camera follow
  - 2D maze visuals
  - 3 Math NPCs per level
  - Question UI (TextMeshPro)
  - 5 levels with in-memory ScriptableObject question databases

How to run:
1) Open the project in Unity.
2) Open SampleScene (Assets/Scenes/SampleScene.unity).
3) Press Play.

Controls:
- WASD / Arrow Keys: move
- E: interact with NPC

Gameplay:
- Answer 3 questions in a row for each NPC.
- Complete all NPCs on the level to go to the next level.
- Total levels: 5.

Important:
- If TextMeshPro asks to import essential resources, click Import.
